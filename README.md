# XDUNewsHubBackEnd
西电新闻通知整合站(后端)

## 使用方法
1. clone this repo

    ```
    git clone https://github.com/hzrr/XDUNewsHubBackEnd.git
    ```

2. activate virtual env

3. install requirements 

    ```
    pip install -r requirements.txt
    ```

4. check database exists and create tables

    ```
    python create_database.py
    ```

5. run app.py

    ```
    python app.py
    ```

    

**python version:** >=3.10
